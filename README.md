# ToBeDone

Cloud-Based Team Collaboration Platform with AI-Powered Notification Digest.
MCA Final Project — Team of 3.

## Tech Stack
- **Frontend:** React (Vite), React Router, Axios, Zustand
- **Backend:** Node.js, Express (ES6 modules), MongoDB (Atlas), Mongoose
- **Realtime:** Socket.io
- **Integrations:** Slack Webhook, AI Digest Agent (Claude/Gemini API)

## Project Structure
ToBeDone/
├── frontend/    → React app
└── backend/     → Express API

## Getting Started

### Backend
```bash
cd backend
npm install
# create .env file (see .env.example)
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
